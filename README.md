# logo2avatar

校徽头像制作小程序，实现不同校徽与头像的叠加，生成新的头像。

使用云开发构建项目，头像制作功能使用小程序的 canvas api。

校徽图片使用爬虫获取的，如果你需要，可以参考这篇文章：[Python 爬取校徽图片](https://github.com/Mayandev/fever_code/tree/master/python/avatar_scrapy)
